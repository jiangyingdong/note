Python 学习 numpy数组
	常用的创建初始化数组的函数:
		numpy.array() #将list or tuple 转化为数组
		numpy.arange(0,7,1,dtype=numpy.int16) # 0为起点，间隔为1时可缺省(引起歧义下不可缺省)
		numpy.ones((2,3,4),dtype=numpy.int16) # 2页，3行，4列，全1，指定数据类型
		numpy.zeros((2,3,4)) # 2页，3行，4列，全0
		numpy.empty((2,3)) #值取决于内存
		numpy.arange(0,10,2) # 起点为0，不超过10，步长为2
		numpy.linspace(-1,2,5) # 起点为-1，终点为2，取5个点
		numpy.random.randint(0,3,(2,3)) # 大于等于0，小于3，2行3列的随机整数
		
	乱序:
		使用random.shuffle(list)函数:
			若是打乱数组第一个维度的顺序,可以将数组转化为list,然后打乱,再转回array:
				list1 = list(numpy.ones((3,4)))
				random.shuffle(list1)
				array = numpy.array(list1)
			若是有目标的打乱则可以使用循环:
				list2 = list3 = list(range(10,20)) #目标是以相同的顺序打乱这两个list
				list1 = list(range(len(list2)))
				random.shuffle(list1)
				print(list1)
				new_list2 = list()
				new_list3 = list()
				for i in list1:
					new_list2.append(list2[i])
					new_list3.append(list3[i])
				print(new_list2,new_list3)
